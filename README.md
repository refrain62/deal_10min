# deal_10min
「これが10分でできない人は自分がかなりプログラミングができないというつらい事実を認識しましょう。」だそうな。  
人数とカードが与えられたとき、分配後のカードを配列として返す関数を作成せよ。  
カードは与えられた文字列の前から順番に配られ、配られるカードの枚数は均等で余りは無視することとする。
